## Troubleshooting

### Can't receive emails:
- Check incoming server settings (IMAP/POP)
- Verify port numbers (usually 993 for IMAP SSL)
- Check spam/junk folders

### Can't send emails:
- Check outgoing server settings (SMTP)
- Verify port numbers (usually 587 for TLS or 465 for SSL)
- Some networks block port 25

### Sync issues:
- Switch from POP to IMAP
- Check "Sync" settings in app
- Restart the app/device

## Security Best Practices

1. **Use strong, unique passwords** for each email account
2. **Enable 2-Factor Authentication** where available
3. **Use a password manager** (1Password, LastPass, etc.)
4. **Don't access email on public WiFi** without VPN
5. **Regularly review connected apps** and revoke unused access
