### Pre-Submission Requirements
- [ ] Android App Bundle (.aab) built and signed
- [ ] App tested on multiple Android devices
- [ ] Feature graphic (1024 x 500)
- [ ] App icon (512 x 512, 32-bit PNG)
- [ ] Screenshots (minimum 2, maximum 8):
  - Phone: 16:9 or 9:16 aspect ratio
  - Tablet: Additional tablet screenshots (optional)
- [ ] Privacy policy URL
- [ ] App description (max 4000 characters)
- [ ] Short description (max 80 characters)
- [ ] Title (max 50 characters)
- [ ] Promotional video (optional, YouTube link)

### Play Console Configuration
- [ ] Create app in Google Play Console
- [ ] Select app category
- [ ] Set content rating (IARC questionnaire)
- [ ] Configure pricing and distribution
- [ ] Set up in-app products/subscriptions
- [ ] Add app content details
- [ ] Configure target audience and content
- [ ] Set up data safety section

### Technical Requirements
- [ ] Target Android SDK 33+ (Android 13)
- [ ] 64-bit native libraries included
- [ ] App permissions declared in manifest
- [ ] Privacy policy accessible and compliant
- [ ] SSL/TLS for all network communications
- [ ] ProGuard/R8 obfuscation enabled

### Data Safety Section (Required)
- [ ] Data collection practices documented
- [ ] Security practices described
- [ ] Data sharing with third parties disclosed
- [ ] User data deletion option available

### Store Listing Assets
- [ ] High-res app icon (512 x 512)
- [ ] Feature graphic (1024 x 500)
- [ ] Phone screenshots (minimum 2)
- [ ] Tablet screenshots (if applicable)
- [ ] Promotional video (recommended)

## üîê DATA SAFETY REQUIREMENTS

**Google Play requires disclosure of:**

1. **Data Collection**
   - What data is collected
   - How data is used
   - How data is shared

2. **Security Practices**
   - Data encryption in transit (SSL/TLS)
   - Data encryption at rest
   - User can request data deletion
   - Follows Families Policy (if applicable)

3. **Data Types**
   - Personal info (name, email, phone)
   - Health and fitness data
   - Photos and videos
   - Location data
   - App activity
