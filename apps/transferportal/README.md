# Transfer Portal - Athlete Transfer Management

**Domain:** transferportal.ai

## Purpose
Dedicated platform for managing athlete transfers across colleges and sports programs with AI-powered matching and analytics.

## Features
- Transfer portal listings
- AI-powered school matching
- Transfer wizard guidance
- School profiles and statistics
- Transfer analytics and insights

## Deployment
- **Primary Domain:** transferportal.ai
- **Platform:** Netlify
- **Build Command:** `pnpm install && pnpm run build`
- **Publish Directory:** `dist`

## Environment Variables
See `.env.example` for required configuration.
