# Dozier Holdings Group (DHG) - Master Site

**Domain:** dozierholdingsgroup.com

## Purpose
The DHG master site serves as the parent company website for Dozier Holdings Group, showcasing the entire ecosystem of companies and applications under the DHG umbrella.

## Features
- Company overview and mission
- Portfolio of companies and apps
- Investor information
- Contact and partnerships
- Links to all subsidiary apps

## Deployment
- **Primary Domain:** dozierholdingsgroup.com
- **Platform:** Netlify
- **Build Command:** `pnpm install && pnpm run build`
- **Publish Directory:** `dist`

## Environment Variables
See `.env.example` for required configuration.
